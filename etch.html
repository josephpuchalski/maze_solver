Joey P's Path Finder

Instructions

In the Control Panel on the right side, please type a grid size and click "Create Grid" (Recommended Sizes are 70, 80, 90, 100).

You can generate a grid simply by clicking "Fill Grid." You can keep clicking this until you find a grid pattern that you enjoy.

After you are happy with the pattern, you can place the start point by hovering over a grid block and typing the letter "S". This will place a green colored "Start" Square.

Now all you have to do to calculate the shortest path to anywhere you click is simply hover over a different square and type "E". This will place a red colored "End" Square and you can watch the magic begin! You can now simply type "E" over different areas and it will recalculate the path.

If you are feeling crafty, you can also design your own pattern. You can toggle the black squares by either clicking on them or typing "B". Enjoy and Have Fun!

Background

A path finding JavaScript program which utilizes the A* Algorithm to calculate the shortest path. The program also uses the same A* Algorithm to generate its own complex pathways for traversal demonstration. I wrote the program in mind to be easily integrated into other apps for point and click behavior with collision avoidance.
